## 0. Setup

1. Open terminal
2. Login into IBM Cloud CLI:

	```
	ibmcloud login
	```
3. Configure the CLI:

	```
	ibmcloud target --cf
	```
4. Clone repo:

	```
	git clone https://github.com/eciggaar/go-serverless-with-java.git
	```
5. Change directory to cloned repo:

	```
	cd go-serverless-with-java
	```
